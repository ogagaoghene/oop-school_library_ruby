### Setup

### Clone this repository

```bash
git clone https://github.com/ogagaoghene/oop_school_library_ruby.git
```
### Move into the directory with

```bash
cd oop_school_library_ruby
```

### Run Project
```bash
./main.rb
```

### Install linter

```bash
bundle install
```

### Run linter

```bash
rubocop .
```

### Auto-correct

In auto-correct mode, RuboCop will try to automatically fix offenses:

```bash
rubocop -A
```

```bash
rubocop --auto-correct-all
```
### Run Tests
``.
rspec spec/
or 
rspec spec/book_spec.rb
rspec spec/classroom_spec.rb
rspec spec/person_spec.rb
rspec spec/rental_spec.rb
rspec spec/student_spec.rb
rspec spec/teacher_spec.rb
```
